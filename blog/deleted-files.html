Files Accidentally Deleted — How to Recover Them

⊞ Windows Mac June 19, 2025 · 5 min read By CAER Technologies

Deleting a file by accident is one of the most stressful moments in everyday computing — especially when it's something important. The good news: in most cases, your file is not actually gone. Windows and Mac both have built-in recovery options, and if you act quickly and follow the right steps, you can get it back without paying for third-party software.

⚡ Most important rule: Stop using your computer as much as possible until you recover the file. Writing new data to your drive can permanently overwrite deleted files before you retrieve them.

Step 1 — Check the Obvious Places First

Before trying anything advanced, make sure the file isn't simply hiding somewhere you haven't checked.

⊞ Windows — Check the Recycle Bin
  1. Double-click the Recycle Bin icon on your desktop.
  2. Look for your file — you can sort by Date Deleted to find recent deletions.
  3. Right-click the file and choose Restore. It goes back to its original location automatically.
Mac — Check the Trash
  1. Click the Trash icon in your Dock.
  2. Find your file — right-click it and choose Put Back.
  3. The file returns to where it was before deletion.

Step 2 — Check Cloud Storage

If the file was in a synced folder — OneDrive, iCloud Drive, Google Drive, or Dropbox — check the cloud trash. Most services keep deleted files for 30 days or more.

⊞ OneDrive Recycle Bin
  1. Go to onedrive.live.com and sign in.
  2. Click Recycle bin in the left sidebar.
  3. Find your file, right-click it, and choose Restore.
iCloud Drive
  1. Go to icloud.com and sign in.
  2. Scroll to the bottom of the main page and click Data Recovery, then Restore Files.
  3. Find your file and click Restore.

Step 3 — Use Built-In Backup & Version History

If the file isn't in the Recycle Bin or cloud trash, your operating system's backup tools may have a saved copy.

⊞ Windows — File History & Previous Versions
  1. Navigate to the folder where the file used to be.
  2. Right-click the folder and choose Restore previous versions.
  3. A list of snapshots appears — pick one from before the file was deleted and click Open to browse it.
  4. Drag the file out to your desktop to restore it.

Note: This only works if File History or Windows Backup was enabled beforehand.

Mac — Time Machine
  1. Open the folder where the file lived.
  2. Click the Time Machine icon in your menu bar (or search for it in Spotlight).
  3. Use the timeline on the right to go back to a date before the deletion.
  4. Find your file and click Restore.

Note: Time Machine must have been set up and connected to a backup drive.

Step 4 — If None of the Above Worked

If the file wasn't in the Recycle Bin, wasn't in a cloud folder, and no backup existed — you may still have options, but this is where things get more technical.

⚠️ Stop writing to the drive immediately. Every new file saved, every program opened, every website visited can overwrite the space where your deleted file lives. The sooner you act, the better your chances.

At this point your best options are:

  • Reputable free recovery tools — Recuva (Windows) and Disk Drill (Mac/Windows) can scan for recoverable files. Use a free version before paying for anything.
  • Call a professional — If the file is critical (financial records, legal documents, irreplaceable photos), stop everything and call us. Attempting DIY recovery incorrectly can make professional recovery harder or impossible.

How to Prevent This From Happening Again

  • Enable File History on Windows or set up Time Machine on Mac — both are free and built in.
  • Use a cloud storage folder (OneDrive or iCloud Drive) as your primary save location for important work.
  • Consider a proper business backup solution if you're relying on this data for your livelihood.
