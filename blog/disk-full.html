"Your Disk Is Almost Full" — What It Means and What to Delete Safely

⊞ Windows Mac June 16, 2025🕐 5 min read
💾 storage ⚡ performance 🔧 troubleshooting ⭐ beginner

💾 Safe on any OS — do this first

Empty your Recycle Bin or Trash. Files you've deleted are still sitting on your drive until you empty it. Right-click the Recycle Bin (Windows) or Trash (Mac) and choose "Empty." This can instantly free gigabytes with zero risk.

⊞ Windows 10 & 11 storage cleanup steps.
✅ Safe to delete on Windows: Recycle Bin, Downloads folder, temp files (%temp%), old Windows Update files, and apps you no longer use.
⛔ Never delete: Anything in C:\Windows, C:\Program Files, or files ending in .dll or .sys.

Step 1 — Run Disk Cleanup

1

Use the built-in Disk Cleanup tool

Click Start, search Disk Cleanup, and select your C: drive. Check every box in the list. Then click "Clean up system files" and check those boxes too — this also removes old Windows Update files which can be several GB. Click OK and let it run.

Step 2 — Clear Temporary Files

2

Delete the temp folder manually

Press Win + R, type %temp%, and press Enter. Select everything (Ctrl + A), then delete. Some files in use can't be deleted — skip those. Then go to Settings → System → Storage → Temporary files for additional cleanup options with checkboxes.

Step 3 — Use Storage Sense

3

Let Windows identify the biggest space users

Go to Settings → System → Storage. Click each category to see what's taking up space: Apps & features, Temporary files, Videos, etc. You can uninstall apps directly from here by clicking one and selecting Uninstall. Turn on Storage Sense to automate future cleanup.

Step 4 — Clean Your Downloads Folder

4

Delete old installers and downloads

Press Win + E to open File Explorer and click Downloads in the left panel. Click Size at the top to sort largest to smallest. Old .exe and .zip installers are almost always safe to delete once you've already installed the software.

💡 Windows tip: If you use OneDrive, enable Files On-Demand in OneDrive settings. This keeps files in the cloud and only downloads them when you open them, freeing up significant space without losing access to anything.
macOS storage cleanup steps.
✅ Safe to delete on Mac: Trash, Downloads folder, large old files, apps you don't use, and iCloud-backed photos already in the cloud.
⛔ Never delete: Anything in /System, /Library, or /usr — these are macOS system folders. Never use random "cleaner" apps from the internet.

Step 1 — Use macOS Storage Recommendations

1

Let macOS find what to delete

Go to Apple Menu () → About This Mac → More Info → Storage Settings. Click Recommendations. macOS will suggest: storing files in iCloud, optimizing storage for photos, emptying Trash automatically, and reducing clutter. Each recommendation has a direct action button.

Step 2 — Remove Large Files

2

Find your biggest files

In Storage Settings, click Documents → Large Files. macOS shows every file over a certain size, sorted by how big they are. Old video exports, disk images (.dmg files you've already installed), and downloaded movies are often the biggest culprits. Select and delete what you don't need.

Step 3 — Remove Unused Apps

3

Uninstall apps you no longer use

Open Finder → Applications. Drag any apps you don't use to the Trash, then empty it. For a more thorough removal that also deletes preference files and caches, use a free tool like AppCleaner (downloaded from freemacsoft.net — it's reputable and free).

Step 4 — Clear App Caches

4

Delete cached data from apps

In Finder, press Cmd + Shift + G and type ~/Library/Caches. You'll see folders for each app. You can delete the contents of individual app cache folders (not the folders themselves). Browsers like Chrome and Firefox store large caches here — clearing them is always safe.

💡 Mac tip: If you use iCloud Photos, turn on Optimize Mac Storage in Photos → Settings → iCloud. macOS will keep full-resolution photos in iCloud and only store smaller previews locally, potentially freeing tens of gigabytes.

Long-Term Solution

If you've cleaned everything obvious and are still running tight on space, an external hard drive (~$50 for 1TB) or expanding your cloud storage subscription is the most reliable fix. We can also audit your drive remotely to identify hidden system files or software caches that aren't visible through normal cleanup tools.
